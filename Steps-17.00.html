Year slicer

1.     Click on previous slicer “LeadTimeAdjustment” then press “Ctrl+C” to copy, “Ctrl+V” to paste. 
2.     Go to Visualization pane, build visual; 
Field: delete “LeadTimeAdjustment” 
3.     Go to Data pane, expand Calendar and select Year
4.     In the Filter pane: Year, select “is not” from drop down list, then enter “2014” and click on “Apply filter”
5.     Go to Visualization pane, format visual; 
Value: Font= DIN, SIZE=9, bold
6.     Go to General tab, 
Properties: Size: Height = 100, Width = 190, Position: Horizontal = 10, Vertical = 149
Title: title=Replace “LeadTimeAdjustment” with “Year”

Reset Button

1.     Click on title “Repeated Guest” then press “Ctrl+C” to copy, “Ctrl+V” to paste. 
2.     Go to General tab, 
Properties: Size: Height = 79, Width = 190, Position: Horizontal = 10, Vertical = 1
Title: title=Replace “Repeated Guest” with “Reset Dashboard”
3.      Go to format, send to the back.
4.     Go to view, bookmarks; In the bookmarks pane click “Add” then right click “Bookmark 1” rename= “Default Slicer”
5.     Go to Insert, Buttons, Blank. 
6.     In the Format button pan, go to button then style.
Style: state=Default, Text=on, text: Reset, Font= Segoe UI, size= 13, color=666666; Icon=off; Fill=on, color= F9F3EF, Transparency=0%; border=on, color=702254, width=1. 
Style: state=On press, Font color=12239E; Fill=on, color= 702254.
 Action: Action=On, Type=Bookmark, Bookmark=Default Slicer.
7.     Go to General tab, 
Properties: Size: Height = 39, Width = 140, Position: Horizontal = 35, Vertical = 35
