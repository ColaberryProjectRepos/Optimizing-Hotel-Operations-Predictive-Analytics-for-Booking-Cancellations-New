Go to Visualization; Format your report page; Canvas settings: Type = Tooltip
 
2.     Select visual “Aviation”; 
General tab: Properties: Size: Height = 33, Width = 124; Position: Horizontal = 146, Vertical = 15.  
Visualization: Field= Lead Time (First)
 
3.     Select visual “Marker Segment”; 
General tab: Properties: Title= ON, Text = Lead Time
