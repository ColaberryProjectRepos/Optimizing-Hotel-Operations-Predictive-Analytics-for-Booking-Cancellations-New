Creating Calendar table


Detailed Instructions:
  1. In Power BI Desktop, click on Modeling tab and select new table. 
  2. In the formula bar, enter the following DAX formula, click  to commit and  to close

Calendar =

ADDCOLUMNS(

CALENDARAUTO(),

"Year", Year([Date]),

"Quarter", "Q" & FORMAT([Date], "Q"),

"Month Number", Month([Date]),

"Month", FORMAT([Date],"MMM")

)

Creating Refresh Date table



Detailed Instructions:
  1. In Power BI Desktop Home ribbon, click on Get data and select Blank Query. Power Query will open.
  2. Under Queries, right click Query 1 and rename ‘Refresh Date’.
  3. Next, go to the formula bar and type the following DAX code and click  to commit:                       = DateTime.LocalNow()
  4. The current date and time will be displayed. Click on Convert to Table under the Date Time Tools Tab. 
  5. After the refresh table pops up, Right click Rename Column1 as ‘Date time’ 
  6. Click on Close & Apply to save.

LeadTimeAdjustment parameter table





Detailed Instructions:
  1. In Power BI Desktop Home ribbon, go to the 'Modeling' tab and click 'New Parameter'.
  2. Select Numeric Range 
  3. Define the parameter as follows;                                                                                                                Name: LeadTimeAdjustment
          DataType: Decimal Number
          Minimum: -0.2 
          Maximum: +0.21 
          Increment: 0.01  
          Default Value: 0
  4. Click Create, which will automatically create a slicer and a parameter table.
  5. In Data pane, expand LeadTimeAdjustment table and click on LeadTimeAdjustment    column to open the column tool ribbon.
  6. Under Format, select percentage and set the decimal places to ‘0’ then click  to commit and  to close
