Add background image, project logo, colaberry logo and project title 16


Adding background image:  

1.     Go to Visualization pane: Format your report page.
2.     Expand Canvas background: Browse Image and select “dashboard background” from the downloaded file
3.     Set Transparency= 0%
4.     Format LeadTimeAdjustment slicer: click on slicer, go to general tab and set properties:
Size: Height = 73, Width = 190
Position: Horizontal = 10, Vertical = 356

Adding project logo:  

1.     Insert: Image and select project logo from the downloaded file
2.     Go to General tab and set Properties: 
Size: Height = 66, Width = 140
Position: Horizontal = 210, Vertical = 7

See gif below



Adding colaberry logo:  

1.     Insert: Image and select colaberry logo from the downloaded file
2.     Go to General tab and set Properties: 
Size: Height = 66, Width = 180
Position: Horizontal = 1090, Vertical = 7

see gif below



Adding project Title:  

1.     Insert a Text Box and add text “Predicting Hotel Cancellations.”
2.     The Font=Segoe UI, Size=32, Color=785C83, Bold and Center
3.     Go to General tab and set Properties: 
Size: Height = 67, Width = 695
Position: Horizontal = 375, Vertical = 7
