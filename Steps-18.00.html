Avg Lead Time (LT) main KPI

1.     Click on previous slicer “LeadTimeAdjustment” then press “Ctrl+C” to copy, “Ctrl+V” to paste. 
2.     Go to Visualization pane, build visual; and select Card icon
Field: click  to delete “LeadTimeAdjustment” 
3.     Go to Data pane, expand Bookings_Measures and select LT
4.     Click on LT, Measure Tool, Format= whole number
5.     Go to Visualization pane, format visual; Category label=off,
Callout value: Font= Segoe UI, size= 25, color=094780
6.     Go to General tab, 
Properties: Size: Height = 79, Width = 190, Position: Horizontal = 10, Vertical = 249
Title: title=Replace “LeadTimeAdjustment” with “Avg Lead Time (LT)”

See gif below for live steps
