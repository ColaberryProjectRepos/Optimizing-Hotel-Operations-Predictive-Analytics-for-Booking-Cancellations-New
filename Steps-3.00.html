Data Overview 3

Data Sources and KPI:

Data were gathered from the hotels' Property Management System (PMS) databases via Microsoft SQL Server, covering the period from July 2015 to August 2017. This dataset is available on Kaggle, a reputable online data repository.



System Architecture

The system architecture involves understanding business needs to develop targeted cancellation prevention strategies, gathering relevant data, cleansing and preprocessing the data, selecting and training suitable models using Python, analyzing model results, and visualizing insights using Power BI to support decision-making.



Data Features

The dataset includes variables such as booking channels, lead times, customer segments, and special requests among others, which are critical for predicting cancellation rates and optimizing resource allocation.
